# 맛집 공유 및 예약 커뮤니티(yummy)
위치와 키워드를 활용하여 맛집공유 글쓰기와 댓글 쓰기, 예약하기 기능이 가능한 커뮤니티를 제작하려고 합니다.


## 프로젝트 기능
### 1.회원관리 기능

* 회원가입
  	* 회원 아이디 / 비밀번호 / 선호 키워드 / 위치(현위치 or 선택 가능)
* 회원 정보 보기(내정보)
* 회원수정
* 회원탈퇴
